# mamba-threat-intel

Repository scaffold for Mamba Threat Intelligence research and tooling.

Structure

See project tree in the repo root. This scaffold contains starter modules and placeholders.

Quickstart

- Create a virtual environment and install requirements:

  python -m venv .venv; .\.venv\Scripts\Activate.ps1; pip install -r requirements.txt

- Run the dashboard (dev):

  python src/dashboard/app.py
